## 5. ИЗВЛЕЧЁННЫЕ УРОКИ ПО ПРОМПТИНГУ

**Что уводило промпт не туда:**
- «hollow cheeks», «angular», «V-shaped jawline» → лицо становилось
  слишком взрослым, худым, резким. Убрать для юного образа.
- «hooded eyes», «sleepy» → глаза становились слишком узкими и усталыми.
- «platinum / silver / ashy» → волосы становились седыми.
- «golden-blonde / sun-bleached / luminous / warm» → волосы уходили в рыжину/золото.
  Для правильного блонда использовать «light blonde, neutral cool-toned, ash-blonde».
- «dark blonde eyebrows» → слово «dark» цепляло волосы и делало корни
  тёмными. Заменить на «natural blonde eyebrows».

**Что работает:**
- «youthful around 20-21», «soft full cheeks» → держат юность
- «enormous disproportionately oversized lips» → большие губы
- «snake bites» → пирсинг как якорь
- Для Nano Banana — негативы через NOT работают
- Для ChatGPT Images — негативы переписывать в позитив
